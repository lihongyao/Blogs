# 开始

## 创建项目

在终端输入如下指令根据提示构建项目：

```
$ npx create-umi@latest
┌   create-umi 
│
○  What's the target folder name?
│  umi-admin-template
│
○  Pick Umi App Template
│  Ant Design Pro
│
○  Pick Npm Client
│  pnpm
│
○  Pick Npm Registry
│  taobao
│
└  You're all set!
```

> **注意**：请严格按照上述选择项构建项目。
